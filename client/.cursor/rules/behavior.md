# Agent Behavior Rules

## Before Editing

Агент обязан:

1. Найти существующий паттерн
2. Следовать ему
3. Не создавать новый стиль

---

## Decision Making

Если есть 2 варианта:
- выбрать более простой

---

## Code Changes

- Не переписывать файл полностью без причины
- Не ломать существующую структуру

---

## Anti-Patterns

Запрещено:

- создавать лишние файлы
- дублировать компоненты
- использовать any
- добавлять "use client" без причины
- делать fetch в client без необходимости
